# Consistent Overhead Byte Stuffing(COBS) Library (C++)

## How To Use Library
Just copy and paste cobs.cpp and cobs.h to your project. No dependencies.

## How To Run Tests
Tests use testing framework(gtest) by google. Install gtest. Run:  
```
$ cd tests 
$ make
```
